FreechargeBiz Credit Card Advisor
An AI-powered financial companion that provides personalized credit card recommendations by analyzing spending habits via Account Aggregator (AA) data and LLM-driven insights. Designed with the Freecharge visual identity.

🚀 Features
Intelligent Recommendations: Uses an LLM to match spending profiles with card benefits.

Account Aggregator Integration: Connects securely to pull and analyze transactional data.

Data-Driven Logic: Deterministic eligibility filtering (Income, CIBIL, PAN validation) before AI analysis.

Modern UI: A clean, responsive interface built with React, Vite, and Tailwind CSS, featuring Freecharge-inspired design tokens.

🛠 Tech Stack
Backend: FastAPI (Python), Pydantic, Groq API (LLM inference).

Frontend: React, Vite, Tailwind CSS.

Data: Synthetic AA transactional data and structured card catalogs.
